📖 AWS Cloud Survival — Manual

🚦 Traffic Types

  • STATIC – Images, CSS, JS files → route to S3 / CloudFront
  • READ – API data fetches → RDS / DynamoDB / ElastiCache
  • WRITE – Database writes → RDS / DynamoDB / SQS → DB
  • UPLOAD – File uploads → S3 (heavy processing)
  • SEARCH – Complex queries → OpenSearch / RDS only
  • STREAM – Real-time events → Kinesis → Lambda/RDS
  • DDoS – Attacks → block with WAF / Shield
  • AUTH – Login requests → Cognito / API Gateway
  • ML – Inference → SageMaker / Lambda

🔧 Services (26 total)

  • Security: WAF (L7 DDoS/AUTH), Shield (L3/L4 DDoS burst), Cognito (AUTH), GuardDuty (threat detection DD+AUTH), KMS (encryption compliance, all traffic), CloudTrail (audit logging, all traffic)
  • Edge: Route 53 (DNS routing), CloudFront (CDN 70% hit), API Gateway (rate-limits), Global Accelerator (TCP/UDP anycast, all traffic)
  • Networking: ALB (round-robin), NLB (TCP-level, lower latency), VPC NAT (passes all traffic)
  • Compute: EC2 (general), Lambda (fast burst, limited), ECS/Fargate (scalable containers), EKS (Kubernetes)
  • Database: RDS (SQL: READ/WRITE/SEARCH), DynamoDB (NoSQL: READ/WRITE faster), Aurora (high-performance SQL), DocumentDB (document store)
  • Caching: ElastiCache (35–65% hit rate), DAX (DynamoDB Accelerator, READ only)
  • Storage: S3 (STATIC/UPLOAD), EFS (shared file system)
  • Messaging: SQS (buffers 200 req), SNS (fan-out broadcast), Kinesis (STREAM ingestion)
  • AI/ML: SageMaker (handles ML traffic)
  • Search: OpenSearch (SEARCH + READ, high capacity)

🛠 Tools

  • Select – Drag nodes · Click upgradeable nodes to upgrade · Inspect in right panel
  • Link – Click source → click destination to connect
  • Unlink – Click on a connection arrow to remove it

⚙️ Mechanics

  • Traffic spawns from the left edge as color-coded packets
  • Packets route through your graph following connections
  • Unhandled packets lose −1% Reputation; DDoS leaks lose −2%
  • Node health degrades from overload; 0% health nodes hurt reputation
  • Waves escalate every 60s in Survival mode — each wave introduces new traffic types
  • Budget −$1000 or Reputation 0% = Game Over
  • Upgrades: Tier 1→2→3 doubles/triples capacity and speed
  • Shift+click a service card to place multiple instances
  • Scroll wheel to zoom canvas · Middle-drag to pan
  • Hotkeys: S=Select, C=Connect, D=Delete, U=Unlink, Space=Pause, R=Reset view
