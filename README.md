# AWS-Game
🎮 Built an AWS exam prep game — because flashcards weren't cutting it.
I created AWS Cloud Survival, a browser-based strategy game that teaches real AWS architecture through gameplay.

https://samyvishnu.github.io/AWS-Game/

You're an architect under fire. Traffic is spawning, DDoS attacks are incoming, and your infrastructure has to hold. 
Place and connect 30 AWS services — WAF, ALB, Aurora, Kinesis, SageMaker, GuardDuty, KMS and more — into a working architecture before your reputation hits zero.

What makes it different from typical study tools:
•	Each wave escalates with new traffic types: static, reads, writes, uploads, streams, auth floods, ML inference, and DDoS
•	Every wave pauses with a real exam question — tagged SAA-C03, SAP-C02, or SCS-C02 — with a detailed explanation and budget reward if you get it right
•	Security compliance panel tracks 11 checks: PCI-DSS, HIPAA, SOC2, HA, and more in real time
•	Node Inspector shows pricing, anti-patterns, and "when to use" guidance for every service you place — the same decision frameworks tested on the actual exams

47 quiz questions across 16 waves covering the concepts that consistently show up on all three exams: latency routing vs failover routing, DAX vs ElastiCache, KMS CMK vs AWS Managed Keys, Shield vs WAF layers, SCPs vs IAM permission boundaries, Kinesis vs SQS, Global Accelerator vs CloudFront, and more.

Built entirely in vanilla HTML/CSS/JS — single file, no dependencies, runs offline.

Studying for AWS certifications? Drop it in a browser and start building.
